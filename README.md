Projet Pikachu – Synthèse d’Image (OpenGL)

Ce projet a été réalisé dans le cadre de ma troisième année de Licence Informatique (L3) à l’Université de Bourgogne, pour la matière Synthèse d’Image.

L’objectif du projet était de modéliser et afficher un Pikachu en 3D à l’aide de OpenGL, en utilisant des formes géométriques simples (cylindres, sphères, etc.) et des transformations pour composer un personnage complet.

Objectifs du projet

Découvrir et manipuler OpenGL en C++

Apprendre à construire un modèle 3D à partir de formes basiques

Gérer :

transformations (translation, rotation, mise à l’échelle)

lumières

couleurs et textures

Créer un rendu visuel cohérent d’un personnage (Pikachu)

Contenu du projet

Le projet contient :

Des fichiers C++ (classes et gestion des formes)

Le modèle de Pikachu construit à partir de primitives géométriques

Les ressources nécessaires (images, PDF, etc.)

Le fichier main.cpp pour lancer l’affichage OpenGL

Technologies utilisées

C++

OpenGL

GLUT / FreeGLUT
